## Skeleton for Minetest

### Description:

A skeleton mob using the [cmer (Creatures Revived)](https://forum.minetest.net/viewtopic.php?t=26684) or [mobs_redo](https://content.minetest.net/packages/TenPlus1/mobs/) API.

### Requirements:

```
Depends: none

Optional depends:
  - mobs
  - cmer
  - cmer_zombie
  - creatures
  - zombie
  - asm_spawneggs
  - sounds
```
